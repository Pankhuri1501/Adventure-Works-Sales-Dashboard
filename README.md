# Adventure-Works-Sales-Dashboard
Interactive Power BI Sales Dashboard built using Adventure Works dataset to analyze revenue, profit, orders, returns, customers, products, and regional performance. Features KPI tracking, forecasting, drillthrough navigation, dynamic filters, customer insights, product analysis, and return trend monitoring using DAX and Power Query.


# 🚴 Adventure Works Sales Dashboard | Power BI Project

## 📌 Overview
This project is an interactive **Power BI Sales Dashboard** developed using the **Adventure Works dataset** to analyze business performance across sales, customers, products, returns, and regions.

The dashboard enables stakeholders to monitor KPIs, identify trends, and make data-driven decisions through interactive visualizations and dynamic filtering.

# 📈 Key Business Insights

- Accessories generated the highest number of orders.
- Tires and Tubes was the most ordered product category.
- Shorts had the highest product returns.
- Revenue showed significant growth after 2021.
- Professional occupation customers contributed the most orders.
